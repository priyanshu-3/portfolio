# Portfolio Website

Modern, professional portfolio website for Priyanshu Mehra showcasing projects, skills, and achievements.

## Features

- Modern dark theme with vibrant gradients and glassmorphism effects
- Smooth scroll animations, hover effects, and micro-interactions
- Interactive 3D resume viewer with scroll animation
- Fully responsive across desktop, tablet, and mobile
- SEO optimized with proper meta tags and semantic HTML

## Live Demo

Visit the live portfolio: [priyanshu.online](https://www.priyanshu.online)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- EmailJS

## Sections

1. **Hero** - Introduction with social links
2. **About** - Background and statistics
3. **Skills** - Technical skills organized by category
4. **Projects** - Featured projects with details
5. **Resume** - Interactive PDF viewer with 3D scroll animation
6. **Contact** - Contact form with email integration

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/priyanshu-3/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the dev server:
```bash
npm run dev
```

4. Open in browser:
```
http://localhost:5173
```
