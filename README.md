# Portfolio Website

Modern, professional portfolio website for Priyanshu Mehra showcasing projects, skills, and achievements.

## 🌟 Features

- **Modern Design**: Dark theme with vibrant gradients and glassmorphism effects
- **Smooth Animations**: Scroll animations, hover effects, and micro-interactions
- **3D Resume Viewer**: Interactive PDF resume with scroll animation
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile
- **SEO Optimized**: Proper meta tags and semantic HTML

## 🚀 Live Demo

Visit the live portfolio:[[ ](https://portfolio-azure-tau-i5qw5xmjci.vercel.app/)](https://portfolio-azure-tau-i5qw5xmjci.vercel.app/)

## 📁 Project Structure

```
portfolio/
├── index.html              # Main HTML file
├── style.css               # Styling and animations
├── script.js               # Interactive features
├── resume.pdf              # Resume PDF
├── blockchain-project.jpg  # Project image
└── skincare-project.jpg    # Project image
```

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Inter)

## 📱 Sections

1. **Hero** - Introduction with social links
2. **About** - Background and statistics
3. **Skills** - Technical skills organized by category
4. **Projects** - Featured projects with details
5. **Achievements** - Awards and certifications
6. **Resume** - Interactive PDF viewer with 3D scroll animation
7. **Contact** - Contact information and links

## 🎨 Key Features

- **3D Scroll Animation**: Resume section features perspective transforms
- **Cursor Trail Effect**: Interactive cursor trail on desktop
- **Project Card Tilt**: 3D tilt effect on project cards
- **Counter Animations**: Animated statistics
- **Parallax Effects**: Gradient orbs with parallax scrolling
- **Smooth Navigation**: Smooth scroll to sections

## 🚀 Local Development

1. Clone the repository:
```bash
git clone https://github.com/priyanshu-3/portfolio.git
cd portfolio
```

2. Start a local server:
```bash
python3 -m http.server 8000
```

3. Open in browser:
```
http://localhost:8000
```
